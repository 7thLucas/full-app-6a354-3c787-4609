What It Does

Core Features

Casual Player
🎮

Interactive Board

Responsive 3×3 grid — players alternate turns placing X and O marks.

🏆

Win Detection

Automatic detection of all 8 winning combos — rows, columns, and diagonals.

🤝

Draw Detection

Declares a draw automatically when the board fills with no winner.

📊

Score Tracker

Tracks cumulative wins per player across multiple rounds.

🔄

New Game / Reset

One-click restart to begin a fresh game at any point.

👥

Local Multiplayer

Two players share one device — no internet connection needed to play.
